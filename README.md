# Contracting Website 🤝
## Our page for brand establishment and customer inquiries 📃
This website establishes our mission statement, who we are, and opens invitations for inquiries

## Installation Instructions 🚀
1. git pull (to get the latest changes)
2. Install virtual environment for Mac: python3 -m venv venv
3. Activate it: source venv/Scripts/Activate (should see a green venv tag on the CLI)
4. install dependencies: pip install -r requirements.txt
5. Run the website: python app.py
6. Open http://127.0.0.1:5000 in your browser 

## Technologies 🛠️
HTML, CSS, Vanilla JavaScript, Python, Flask
